# Maternal–Fetal Health Prediction System (Edge AI)

A low-cost maternal–fetal health monitoring system designed for continuous pregnancy monitoring using **Edge AI**, wearable sensor patches, and IoT technologies. The project aims to enable early detection of maternal and fetal complications while remaining suitable for rural and low-resource environments.

> **Project Status:** 🚧 Software components completed. Hardware prototype is currently under development. Patent filing is in progress.

## Features

* Real-time analysis of maternal and fetal physiological signals.
* Edge AI-based anomaly detection without requiring continuous internet connectivity.
* Predictive risk assessment for fetal distress and preterm labor.
* Mobile application for real-time monitoring and alert notifications.
* Cloud integration for remote healthcare access and historical data analysis.
* Personalized baseline learning to reduce false alarms.

## Software Stack

* Python
* Flask
* Machine Learning (XGBoost)
* ESP32 (target deployment)
* Bluetooth Low Energy (BLE)
* Cloud Database
* HTML/CSS/JavaScript (Web Dashboard)

## System Workflow

1. Physiological sensor data is collected from wearable patches.
2. Data is preprocessed and analyzed using a trained ML model.
3. Edge AI predicts potential maternal or fetal health risks.
4. Alerts are generated locally and through the mobile application.
5. Data can be securely uploaded to the cloud for healthcare providers.
